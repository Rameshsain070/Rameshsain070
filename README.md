[![Repository](https://img.shields.io/badge/View_Repo-181717?style=for-the-badge&logo=github)](https://github.com/Rameshsain070/Synapse_AI)

A scalable, modular AI agent platform featuring:
- 🧠 **LangGraph & FastAPI** for multi-agent orchestration
- 📚 **Retrieval-Augmented Generation (RAG)** with semantic search
- 💾 **Long-term semantic memory** via Pinecone
- 🔄 **Streaming LLM responses** with Azure OpenAI
- 🐳 **Containerized deployment** with Docker

**Tech Stack:** Python, FastAPI, LangGraph, Pinecone, Azure OpenAI
